# Report Generation

综合所有阶段产物和审查结论，生成项目交付汇总报告。

## 输入
- 所有阶段产物文件
- 所有 Review 报告
- `state.json`（了解各阶段执行情况）

## 输出格式
- `report.md`

## 报告内容

1. **项目概览**：需求简述、交付范围
2. **流水线执行摘要**：各阶段状态、耗时、重试次数
3. **产物清单**：每阶段产出物及审查结论
4. **关键决策**：架构选型、优先级调整等
5. **遗留问题**：未解决的 issue 和技术债务
6. **统计指标**：代码变更量、测试通过率

## 输出结构

```markdown
# 项目交付报告

## 项目概览
## 流水线摘要
## 阶段产物清单
## 关键决策记录
## 遗留问题
## 统计指标
```
